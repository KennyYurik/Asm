Задача — вывести строку, содержащую отформатированное согласно флагам десятичное число.
Программа запускается с двумя аргументами — форматная строка и само число. Аргументы — это строки (возможно, в кавычках), которые пишутся после имени программы при запуске, если кто не знает. Разделяются пробелом, подаются нам уже распарсенными, то есть нам этим заниматься не нужно.
Форматная строка имеет вид: [флаги][длина].
Значение длины — от 1 до 50. Может быть не указано, в этом случае считаем её равной единице. Длина строки результата должна быть не меньше длины, указанной в форматной строке. То есть недостающие символы мы дописываем тем, чего от нас требуют флаги (пробелами или нулями) слева или справа в зависимости от флагов.
Флаги — это набор символов из множества { ' ', '+', '-', '0' }. Флаги могут повторяться. При отсутствии флагов число форматируется так: перед отрицательным числом ставим '-', перед положительным не ставим ничего, выравниваем пробелами по правому краю (поправьте, если ошибаюсь).
Выравнивание: если длина числа меньше требуемой, то добавляем в строку выравнивающие символы (пробелы или нули в зависимости от флагов). Если требуется выровнять число справа, то выравнивающие символы добавляем слева (логично, тогда число останется справа), если требуется выровнять слева, то символы добавляем справа, пока длина строки не станет равна требуемой.
Флаг ' ': для отрицательного числа перед знаком печатать '-', для положительного печатать ' '.
Флаг '+': для отрицательного числа перед знаком печатать '-', для положительного печатать '+'. Этот флаг подавляет флаг ' '.
Флаг '-': выравнивать число не правому, а по левому краю.
Флаг '0': выравнивать число нулями. Подавляет '-'.
Число — строка, содержащая шестнадцатеричное представление 128-битного числа (то есть максимум может быть 32 цифры). Если в строке меньше 32 цифр, значит, в недостающих цифрах — нули. Число в дополнительном коде. В строке перед числом может стоять минус, который меняет знак числа.

Примеры (после символа '>' идёт ввод, после символа '<' — вывод. Символ ' ' заменён на '_' для того, чтобы мерзкий вконтакт не заменял n пробелов на один):
>3 5
<__5

>1 15
<15

>0+5 5
<+0005

>0_5 5 //чтобы нормально скушалось командной строкой, строку формата нужно заключить в кавычки
<_0005

>_ 2
<_2

> 2 //нет флагов
<2

>-3 5
<5__

>05 5
<00005

>-05 5
<00005
